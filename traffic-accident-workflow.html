📋 案件办理全流程

1

接案录入

流程起点

录入当事人信息、事故时间地点、分配主办律师，系统自动生成案件编号（如 2026-JT-06-001）。

当事人姓名 身份证号 事故时间 事故地点 承办律师
📝 自动生成：授权委托书
⏰ 飞书自动提醒
  • 接案当日：提醒联系交警，了解事故认定书进度
2

等事故认定书

AI 识别

上传《道路交通事故认定书》和保单扫描件，系统自动识别被告信息、责任划分、事故类型、认定机关，并自动创建对方当事人和保险公司档案。

事故认定书扫描件 保单 事故类型 对方责任 预估伤残等级
🔍 特色：事故认定书 + 保单 OCR 识别
⏰ 飞书自动提醒
  • 认定书 T+7：提醒确认进度
  • 认定书 T+15：提醒了解原因并催促
3

个人鉴定

诉前鉴定

当事人进行伤残等级鉴定。系统记录鉴定所、法医、鉴定报告及伤残等级，作为后续调解或诉讼依据。

伤残等级 附加等级 鉴定报告 伤情诊断 法医
📝 自动生成：鉴定申请书
⏰ 飞书自动提醒
  • 鉴定前 7 天：提醒通知当事人准备材料
  • 鉴定后 7 天：提醒跟进鉴定报告出具进度
4

定级分流

决策节点

根据伤残等级和案件情况，律师选择「调解」或「诉讼」方向。系统给出建议标签，但最终决策权在律师。

推进方向 调解 / 诉讼
调解方向
A

调解准备

确定调解日期和参与人员，系统可自动计算赔偿金额作为调解方案参考。

🧮 可调用：赔偿金额计算
⏰ 飞书自动提醒
  • 调解前 3 天：提醒准备赔偿方案与材料包
B

调解结果

记录调解成功或失败。调解成功可进入结案；调解失败可转诉讼方向继续推进。

调解结果 赔偿金额 调解书
诉讼方向
C

诉讼准备

填写管辖法院，准备起诉材料。系统根据事故认定书、鉴定结论、赔偿计算结果自动生成民事起诉状等文书。

📝 自动生成：民事起诉状、所函、追加被告申请书、变更诉讼请求申请、授权委托书 🧮 可调用：赔偿金额计算
⏰ 飞书自动提醒
  • 诉讼材料截止前 3 天：提醒检查起诉状与证据清单
D

立案

向法院提交立案材料，记录立案日期和受理通知书。

📝 自动生成：送达地址确认书、诉讼费用退付收款账户确认书、收件清单、证据提交清单
⏰ 飞书自动提醒
  • 立案后每 15 天：提醒确认法院是否已受理
E

法鉴鉴定

法院委托的司法鉴定。记录法鉴等级、附加等级、法医及鉴定报告。

F

受理后排期

法院受理后安排开庭。记录开庭日期和案号，系统自动同步飞书日历提醒。

⏰ 飞书自动提醒
  • 受理后每 15 天：提醒确认开庭传票/排期情况
G

开庭前准备

整理证据清单要点和庭审大致内容。系统可解析已上传 PDF 证据，自动生成补充证据清单。

📝 自动生成：补充证据清单
⏰ 飞书自动提醒
  • 开庭前 3 天：提醒变更诉请或补充材料截止
H

开庭结束等文书

记录收到判决书日期、赔偿金额、医疗费、一审判决文件。

⏰ 飞书自动提醒
  • 开庭后每 15 天：提醒联系书记员确认判决书/调解书进度
I

是否上诉

一审判决后决策：不上诉则判决生效；上诉则进入二审程序。

⏰ 飞书自动提醒
  • 判决书 T+1：提醒确认是否上诉（上诉期 15 天）
  • 判决书 T+3：⚠️ 极高优先级提醒，上诉期即将截止
5

二审程序（如上诉）

可选分支

二审诉讼准备 → 二审立案 → 二审受理后排期 → 二审开庭前准备 → 二审开庭 → 二审开庭结束等文书。流程与一审类似，系统自动生成对应阶段文书和提醒。

6

判决生效

执行起点

记录判决生效日期，案件进入执行阶段。

7

执行

回款阶段

记录执行结果、执行到位金额、执行文书。执行成功或失败均可进入结案确认。

执行结果 执行到位金额 执行文书
⏰ 飞书自动提醒
  • 执行每 15 天：提醒确认回款情况或联系执行法官
8

结案确认

流程终点

填写代理费，确认赔偿金额、医疗费、其他金额。结案后案件进入只读模式，不可再修改。

代理费 赔偿金额 医疗费 其他金额

💬 飞书 + AI 深度集成

律师无需打开 Web 后台，在飞书里用自然语言就能完成大部分案件操作，AI 全程以「律师助理」身份协助。

🔎 自然语言查询案件

在飞书直接发问，AI 自动查询并回复案件进度。

  1. 「张三的案子到哪一步了？」
  2. 「李四的案件鉴定报告出来了吗？」
  3. 「我今天有什么待办？」
  4. 「本周有哪些开庭？」

🚀 自然语言推进阶段

用一句话更新案件阶段，AI 会发送确认卡片，律师确认后系统自动执行。

  1. 「张三的案件事故认定书已经拿到了，推进到鉴定阶段」
  2. 「李四的案子调解成功了，结案」
  3. 系统生成确认卡片，展示当前阶段 → 目标阶段
  4. 点击「确认推进」后自动流转并创建下一阶段提醒

📎 自然语言上传文件

在飞书发送文件并附带说明，AI 自动识别文件类型和当事人，归档到对应案件。

  1. 发送文件并配文「这是张三的事故认定书」
  2. AI 识别文件类型（事故认定书 / 鉴定报告 / 保单等）
  3. 自动匹配当事人「张三」的对应案件
  4. 上传至坚果云并按分类归档到案件详情

📝 自然语言生成文书

在飞书一句话生成文书，AI 自动调用后台生成并返回审批卡片。

  1. 「帮我生成张三案件的民事起诉状」
  2. 系统自动聚合案件数据并生成文书
  3. 飞书返回文书列表卡片，状态为「待审批」
  4. 律师回复「全部确认」或单独驳回/重新生成

📅 开庭日程自动同步飞书日历

记录开庭、调解、鉴定日期后，系统自动创建飞书日程并提前提醒。

  1. 在「受理后排期」「调解准备」等阶段填写日期
  2. 系统自动在律师飞书日历创建日程
  3. 自动设置提醒：开庭前 3 天、调解前 3 天、鉴定前 7 天 + 1 天
  4. 案件阶段变更时自动更新或删除关联日程

✅ 飞书卡片确认一切操作

关键操作不直接执行，均通过飞书卡片让律师确认，避免误操作。

  1. 阶段推进确认卡片
  2. 文书审批 / 驳回 / 重新生成卡片
  3. 提醒「已处理」卡片
  4. 文件类型/案件选择卡片

✨ 三大特色功能

🔍 事故认定书智能识别

上传《道路交通事故认定书》和保单后，系统自动提取关键信息，无需手动录入。

  1. OCR 提取 PDF 文本
  2. AI 识别被告信息（姓名、身份证号、电话、地址）
  3. 识别保险公司名称，自动查询工商信息
  4. 识别事故类型（机动车 vs 非机动车/行人）
  5. 识别对方责任划分（全责/主责/同责/次责）
  6. 自动计算责任比例，同步创建对方当事人和保险公司档案

🧮 赔偿金额自动计算

根据地区赔偿标准、伤残等级、责任比例、当事人信息，自动计算赔偿清单。

  1. 导入地区赔偿标准 Excel（含城镇收入、赔偿限额、责任比例等）
  2. 读取当事人年龄、行业、伤残等级
  3. 计算医疗费、营养费、住院伙食补助、护理费、交通费等
  4. 计算误工费（按实际收入或行业平均工资）
  5. 计算残疾/死亡赔偿金、被扶养人生活费、精神损害抚慰金
  6. 按责任比例折算最终赔偿金额，区分交强险/商业险/超出部分

📝 起诉状自动生成

进入「诉讼准备」阶段后，系统自动生成民事起诉状等诉讼文书，律师审批后即可使用。

  1. 聚合案件数据（当事人、律师、律所、事故信息）
  2. 从事故认定书提取责任划分，从鉴定阶段提取伤残等级
  3. 从赔偿计算结果提取诉讼请求金额和赔偿清单
  4. 使用 Word 模板渲染生成民事起诉状
  5. 同步生成所函、追加被告申请书、变更诉讼请求申请、授权委托书
  6. 文书进入「待审批」状态，律师确认或驳回后可重新生成
💡 律师须知：系统自动生成和计算的结果仅作为工作辅助，最终法律文书和赔偿主张仍需律师审核确认。系统不会在对外沟通中向当事人提供具体赔偿金额预测或法律建议。

📄 文件识别与上传逻辑

1

飞书自然语言上传文件

律师在飞书中发送文件并附带一句话说明（如「这是张三的事故认定书」），AI 自动识别文件类型和当事人，无需手动选择案件。

2

AI 识别文件类型和当事人

系统根据文件名和附带文字，自动判断文件属于哪一类（事故认定书、鉴定报告、起诉状、判决书等），并提取当事人姓名。

事故认定书 鉴定报告 保单 起诉状 判决书 调解书 证据清单
3

关联案件并上传坚果云

系统自动匹配到对应案件，上传文件到坚果云存储，并在案件详情页按分类归档。如果无法确定案件或文件类型，系统会发送卡片让律师选择。

📑 文书自动生成逻辑

⚙️ 触发方式

  1. 系统自动触发：阶段推进到特定阶段后，Agent 收到 document_generate 指令
  2. 律师主动请求：在飞书中说「帮我生成张三案件的起诉状」
  3. Web 端操作：在案件详情页「文书」Tab 手动触发

📋 阶段-文书映射

  1. 接案录入：授权委托书
  2. 个人鉴定：鉴定申请书
  3. 诉讼准备：民事起诉状、所函、追加被告申请书、变更诉讼请求申请、授权委托书
  4. 立案：送达地址确认书、诉讼费用退付收款账户确认书、收件清单、证据提交清单
  5. 开庭前准备：补充证据清单

✅ 审批流程

  1. 文书生成后状态为「待审批」
  2. 律师可逐份确认、驳回或全部确认
  3. 驳回时记录原因，可重新生成
  4. 审批通过后文书正式生效，可下载使用

⏰ 提醒规则说明

系统内置 13 类关键节点提醒，通过飞书推送给主办律师及助理。紧急提醒同时推送短信。

🎯 提醒触发方式

  1. 案件创建时：自动生成接案当日提醒
  2. 阶段推进后：自动生成下一阶段的提醒
  3. 手动创建：律师可针对个案添加一次性提醒
  4. 循环提醒：立案、受理、开庭后、执行阶段按固定周期重复提醒

🚨 优先级规则

  1. 普通：一般跟进提醒（飞书消息）
  2. 高：诉讼材料截止、开庭前准备等（飞书消息）
  3. 紧急/极高：上诉期截止等（飞书 + 短信）
  4. 同一规则在同一案件上最多 3 条未处理提醒，避免重复轰炸

✔️ 停止提醒方式

  1. 点击飞书消息卡片「已处理」
  2. 回复「已处理」「完成」「OK」等关键词
  3. 在 Web 端提醒中心手动关闭
  4. 阶段推进或事项完成后，对应循环提醒停止
